# 🔗 Relay Stats Tracker

> Check your Relay bridge activity instantly — no wallet connect, no signup, 100% free.

Built by [@alpinoWolf](https://twitter.com/alpinoWolf) for the web3 community.

---

## What It Does

Paste any EVM wallet address and instantly see:

- 💰 Total volume bridged in USD
- 📊 Total number of transactions
- ⛓ How many chains you've used and which ones
- ✅ Success rate across all your bridges
- 📋 Full transaction history with from → to chain, amount, date and status
- 📤 Shareable image card to post on X / Twitter

---

## How To Use

1. Visit the live site
2. Paste your wallet address (starts with `0x`)
3. Click **Check Stats**
4. See your full Relay bridge history
5. Share your stats card on X, Telegram or WhatsApp

No wallet connection required. No login. No data stored.

---

## Tech Stack

- Pure HTML / CSS / JavaScript — no frameworks, no dependencies
- Data from [Relay Protocol API](https://relay.link)
- Hosted on GitHub Pages — free forever
